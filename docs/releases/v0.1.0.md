# v0.1.0

Initial working CLI version of Zephyr Workbench.

## Highlights

* YAML architecture models can be validated before rendering
* valid models produce text summaries
* valid models produce Mermaid diagrams
* example architectures are included for secure workplace, identity flow, and enterprise macOS plus Intune plus Windows domain scenarios
* GitHub Actions CI runs tests and validates example files on push and pull request

## Included commands

```bash
python -m zephyr.cli validate <file>
python -m zephyr.cli summary <file>
python -m zephyr.cli diagram <file> --format mermaid
```

## Notes

This release focuses on a stable V1 contract, predictable CLI behavior, and a clear demo path rather than breadth of features.
